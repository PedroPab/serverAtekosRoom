Manual de Usuario

Bienvenido a la plataforma de gestión de proyectos y elementos. Este sistema te permite crear proyectos, organizarlos y vincularles elementos como texto, imágenes, videos y más.

¿Qué es un Proyecto?

Un proyecto es una entidad que actúa como una carpeta donde puedes organizar tus elementos multimedia. Cada proyecto contiene:

  • ID: un identificador único del proyecto.
  • Nombre: un título que define el proyecto.
  • Descripción: un breve resumen sobre el contenido o propósito del proyecto.

¿Qué es un Elemento?

Un elemento es cualquier archivo o contenido multimedia que agregas a un proyecto. Puede ser texto, imagen, video o cualquier otro formato compatible. Lo más importante es que cada elemento está vinculado a un solo proyecto, y no puede existir de manera independiente.

Instrucciones de Uso

1. Crear un Proyecto

Para crear un proyecto, necesitas ingresar un nombre y una descripción. El sistema generará un ID único para el proyecto. Este ID es esencial para poder asociar elementos al proyecto más adelante.

2. Agregar Elementos a un Proyecto

Una vez que has creado un proyecto, puedes comenzar a agregarle elementos. Para hacer esto, selecciona el proyecto utilizando su ID. Los elementos pueden ser de diferentes tipos: texto, audio, video, imágenes, etc.

Nota: El ID del proyecto es obligatorio para asociar elementos al proyecto.

3. Visualizar Proyectos

En la sección de "Visualizar Proyectos", podrás ver la lista de proyectos creados, con su ID, nombre y descripción.

4. Buscar Elementos en un Proyecto

Si deseas ver los elementos asociados a un proyecto, simplemente ingresa el ID del proyecto y obtendrás la lista de elementos vinculados a ese proyecto.
